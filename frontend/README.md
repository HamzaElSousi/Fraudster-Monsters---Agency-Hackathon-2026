# Frontend — Follow The Money

React 19 + Vite frontend for the AI Accountability Dashboard.

## Dev

```bash
npm install
npm run dev       # http://localhost:5173
npm run build     # production build
```

Requires the backend running on `:8000`. Set `VITE_API_URL` in `.env.local` to override.

See the root `README.md` for full setup instructions.
